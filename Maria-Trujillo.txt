    19. What is the difference between a class and an id? - Give me an example of when I might use each one
        You can use a class multiple times, whereas an individual ID should only be used once. You might use a class to change all <p> with the class="blue" to the color blue. An ID would be used if you'd like to change only the last <p> in section blue.
